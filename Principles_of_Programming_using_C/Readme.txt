This repository contains a collection of C programs taught as part of the course
"Principles of Programming using C". The programs are organized by topic and 
demonstrate various fundamental concepts in C programming.

Course Objectives:
------------------
- Understand the basics of the C programming language.
- Learn core programming concepts and logic building.
- Practice writing and running C programs.
- Apply concepts through problem-solving exercises.

How to Compile and Run a Program:
---------------------------------
1. Open a terminal or command prompt.
2. Navigate to the folder containing the `.c` file.
3. Use the GCC compiler:
   gcc filename.c -o output
4. Run the program:
   ./output
